This is a [Next.js](https://nextjs.org) project bootstrapped with [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app).

It has been extended into an AI-powered study planner.  Users can upload a syllabus image, the text is extracted with OpenAI's vision model, and a weekly schedule is generated by the LLM and stored in a Postgres database via Prisma.

## Configuration

Create a `.env` file in the root with the following variables:

```env
# database (Postgres)
DATABASE_URL=postgresql://user:pass@host:5432/dbname?schema=public

# openai API key
OPENAI_API_KEY=sk-...

# Clerk authentication (required for signing in)
# you can obtain both values from the Clerk dashboard "API keys" page.
CLERK_SECRET_KEY=sk_live_...
NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY=pk_live_...
NEXT_PUBLIC_CLERK_FRONTEND_API=your-frontend-api

# redis for caching/queues (optional)
REDIS_URL=redis://localhost:6379
```

Then generate the Prisma client and run the initial migration:

```bash
npx prisma migrate dev --name init
```

## Running Locally

```bash
npm run dev
# or yarn dev, pnpm dev, bun dev
```

Visit [http://localhost:3000](http://localhost:3000) to view the app.

The home page will link you to `/planner` where you can upload a syllabus image and generate a study plan.
